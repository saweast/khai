Программа работы (Planner.exe) с базой данных "Ежедневник".

Демонстрирует использование компонентов ADOConnection, ADODataSet, DataSource, Table и DBNavigator.


База данных Ежедневник (Planer.mdb) представляет собой базу данных в формате Microsoft Access (Microsoft JET), которая состоит из одной единственной таблицы schedule. Таблица schedule, в свою очередь, состоит из двух столбцов aDate (тип DATETIME) и aTask (строковый, 50 символов).

Для того чтобы программа могла работать с базой данных, база данных должна быть зарегистрирована в системе как источник данных ODBC под именем dplanner.

Регистрация:
1. Пуск/Настройка/Панель управления/Администрирование/Источники данных
2. Щелкнуть на кнопке Add
3. Выбрать драйвер: Microsoft Access Driver (*.mdb)
4. Задать:
   Имя источника данных: dPlaner
   Описание: Ежедневник
   База данных: Planner.mdb

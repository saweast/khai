Программа Расходы (spending.exe) обеспечивает работу с базой данных, которая представляет собой текстовый файл (tabl.grd). Для редактирования и просмотра данных используется компонент StringGrid.
Внимание! Каталог проекта называется Spending а не Расходы, так как при испозовании в имени каталога буквы Ы, компилятор не может выполнить компиляцию.

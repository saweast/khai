Программа Погода (meteo.exe) демонстрирует операцию записи в файл - добавляет в файл meteo.txt информацию о температуре воздуха. Если файла данных в текущем каталоге нет, то программа создает его.
Программа Среднемесячная температура (meteoInfo.exe) демонстрирует чтение данных из текстового файла (meteo.txt).

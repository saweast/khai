Программа работы (Stock.exe) с базой данных "Магазин".

Демонстрирует использование компонентов TAble, DataSource, DBGrid, DBEdit, DBMemo.    
Формат базы данных - Paradox. База данных состоит из одной-единственной таблицы stock (stock.db).

Доступ к базе банных осуществляется через BDE.

Для доступа к базе данных необходимо при помощи BDE Administrator создать псевдоним stock (Type: Standard; Default Driver: Paradox). База данных находится в каталоге Database.

Структура таблицы stock.db
-----------------------
Поле     Тип     Размер
------------------------
Title    A        50
Price    $        -
Memo     A        50
Photo    A        30
------------------------

Программа демонстрирует различные методы отображения справочной информации.
В каталоге Chm находятся файлы, необходимые для создания файла справочной системы формата MS HTML Help: файлы справочной информации (*.htm), файл проекта (hhp) и контекста (hhc) справочной системы, а также файл определения идентификаторов разделов (h).
